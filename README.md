# Personal Finance Manager
A menu-driven Personal Finance Manager built using Python and Pickle File Handling

## Features
- Write Transactions
- Read Transactions
- Add Transactions
- Clear All Transactions
- Modify Transaction
- Delete Transaction
- Search Transaction
- Current Transaction
- Income Analysis
- Expense Analysis
- Monthly Report
- Saving Goal Tracker
- Highest Income
- Highest Expense
- Tax calculator

## How to run
1. Install Python 3.13.5
2. Download Finance_Manager.py
3. Run the file:
   python Finance_Manager.py

## Technologies Used
- Python
- Pickle Module
- File Handling
