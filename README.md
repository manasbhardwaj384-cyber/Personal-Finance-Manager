# Personal Finance Manager
A menu-driven Personal Finance Manager built using Python and Pickle File Handling

## Features
-Write Transactions
-Read Transactions
-Add Transactions
-Clear All Transactions
-Modify Transaction
-Delete Transaction
-Search Transaction
-Current Transaction
-Income Analysis
-Expense Analysis
-Monthly Report
-Saving Goal Tracker
-Highest Income
-Highest Expense
Tax calculator

## Technologies Used
-Python
-Pickle Module
-File Handling
